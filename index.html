Camera help (iPhone + Android)

iPhone (Safari)

  1. Open iPhone Settings.
  2. Go to Safari, then Camera.
  3. Set camera access to Allow.
  4. Return to this page and tap Enable camera again.

Android (Chrome)

  1. Open this site in Chrome.
  2. Tap the lock icon near the address bar, then Permissions.
  3. Set Camera to Allow.
  4. Reload the page, then tap Enable camera.

If the preview is still black

  1. Close other apps using the camera.
  2. Switch browser tab away and back, then tap Retry camera.
  3. Make sure this page is loaded from HTTPS.
